GeneratorOf

struct GeneratorOf<T>

A type-erased generator.

The generator for SequenceOf<T>. Forwards operations to an arbitrary underlying generator with the same Element type, hiding the specifics of the underlying generator type.

See also: SequenceOf<T>.
